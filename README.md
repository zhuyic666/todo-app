# 📝 高级待办清单

一个功能完善的待办事项管理应用，支持优先级设置、截止日期提醒和数据导出。

## ✨ 功能特点

- ✅ **添加待办事项** - 快速记录任务
- 🎯 **优先级管理** - 高/中/低三级优先级，颜色区分
- 📅 **截止日期** - 设置截止时间，自动提醒过期任务
- 📊 **智能排序** - 按优先级和截止时间自动排序
- 🎨 **视觉提示** - 过期任务红色警告，今日任务橙色提醒
- 💾 **本地存储** - 数据保存在浏览器，刷新不丢失
- 📤 **导出功能** - 支持CSV和文本格式导出
- 📱 **响应式设计** - 完美适配手机和电脑

## 🚀 快速开始

### 方法一：直接下载使用
1. 下载 `index.html` 文件
2. 双击用浏览器打开即可使用

### 方法二：GitHub Pages部署（推荐）
1. Fork这个仓库
2. 进入 Settings -> Pages
3. 选择 Deploy from a branch
4. 选择 main 分支和根目录
5. 点击 Save，获得访问链接

## 📱 微信使用指南

由于微信内置浏览器的限制，建议：
1. 将HTML文件发送到"文件传输助手"
2. 点击文件 → 右上角"..." → "在浏览器中打开"
3. 选择 Safari（iPhone）或 Chrome（Android）

## 🎨 界面预览

### 优先级颜色
- 🔴 **高优先级** - 红色边框
- 🟡 **中优先级** - 黄色边框
- 🟢 **低优先级** - 绿色边框

### 时间状态
- 🔴 **已过期** - 红色背景
- 🟠 **今天到期** - 橙色背景

## 📤 导出说明

支持两种导出格式：
- **CSV格式** - 包含所有字段，适合Excel打开
- **文本格式** - 格式化文本，便于分享

## 🔧 技术特点

- 纯前端实现，无需服务器
- LocalStorage本地存储
- 响应式布局设计
- 平滑动画效果
- 触摸优化

## 📄 文件说明

- `index.html` - 完整功能的待办清单
- `online-version.html` - 微信优化版本
- `deploy-guide.html` - 微信使用详细指南

## ⚠️ 注意事项

- 数据存储在浏览器本地，清除浏览器数据会删除待办事项
- 建议定期导出重要数据
- 最佳体验：添加到手机主屏幕作为Web应用使用
